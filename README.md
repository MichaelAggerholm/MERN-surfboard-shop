# MERN stack surfboard shop
Projektet går ud på at lave en surfboard shop med ofte anvendte funktionaliteter fra webshops, heriblandt:
1. User authentication
2. Profile section
3. Administration panel
4. Search
5. Filters
6. Google maps
7. Checkout
8. Order overview
9. Reviews
